Parent Functions & Function Notation

▾
🦁 Coach Ibrahim Says

Think of a parent function like Morocco's 4-3-3 base formation — Bounou in goal, Hakimi at right back, El Khanous and El Aynaoui controlling the midfield, Brahim Díaz and Boufal on the wings, Abde Ezzalzouli pressing high. That lineup is the parent. Every tweak is a transformation of it. Szoboszlai doesn't reinvent his game each match — he adapts from his base. Same idea here.

📌 Core Concept

A parent function is the simplest, most basic function in a family. All other functions in that family are transformations of the parent.


FamilyParent FunctionShape
Linearf(x) = xStraight line through origin
Absolute Valuef(x) = |x|V-shape
Quadraticf(x) = x²Parabola (U-shape)
📌 Function Notation — f(x)

f(x) is just a fancy way of writing "y" — it means "the output when you plug in x." You read it as "f of x."

f(x) = 2x + 3 means: take x, multiply by 2, then add 3
🔴 Liverpool — Szoboszlai #8

Szoboszlai's shot accuracy this season is modeled by f(x) = 6x + 5 where x = hours of extra training per week.

1

Find f(3): Plug in x = 3

2

f(3) = 6(3) + 5 = 18 + 5 = 23

3

After 3 hours of training, accuracy score = 23. Makes sense!

🔢 Try This One (from your book)

h(x) = 4x + 3. Find h(10).

1

Replace x with 10: h(10) = 4(10) + 3

2

= 40 + 3 = 43 ✅

💡 Pro Tip — "Is this point on the graph?"

To check if (−1, 2) is on f(x) = 3x + 5:
Plug in x = −1 → f(−1) = 3(−1) + 5 = −3 + 5 = 2. Yes! Output matches. ✅
To check if (−3, −19) is on h(x) = 7x − 2:
h(−3) = 7(−3) − 2 = −21 − 2 = −23 ≠ −19. ❌ NOT on the graph.

📌 Absolute Value Function

f(x) = |x| — absolute value makes everything positive (or zero). Think of it as distance — distance is never negative.


Example: h(x) = |3x − 8|. Find h(1):
h(1) = |3(1) − 8| = |3 − 8| = |−5| = 5

2️⃣

Transformations of Functions

▾
🦁 Coach Ibrahim Says

Transformations are like tactical tweaks on Morocco's 4-3-3. Push Abde Ezzalzouli higher up the wing (translation up). Swap the wingers — put Boufal on the right instead of left (reflection). Tell Hakimi to overlap harder (vertical stretch). Drop El Aynaoui deeper into a double pivot with El Khanous (vertical shrink of the midfield line). The formation is the same parent — you're just transforming it.

📌 The Big 4 Transformations
TransformationWhat ChangesExample
Translation (shift)Moves graph up/down/left/rightf(x) + 3 shifts UP 3
ReflectionFlips graph over an axis−f(x) flips over x-axis
Vertical StretchMakes graph taller/steeper2f(x) doubles height
Vertical ShrinkMakes graph shorter/flatter½f(x) halves height
g(x) = a·f(x − h) + k
k value (outside)

+ k → shifts UP
− k → shifts DOWN

h value (inside)

− h → shifts RIGHT
+ h → shifts LEFT

a > 1

Vertical STRETCH (taller)

0 < a < 1

Vertical SHRINK (flatter)

💥 Liverpool — Darwin Núñez

Núñez's goal rate per intensity level: parent f(x) = |x|
After a training upgrade: g(x) = 2|x − 1| + 3

1

The 2 = vertical stretch — he scores twice as often now (Núñez mentality 🔥)

2

The −1 inside shifts the graph right by 1 (takes 1 extra unit to warm up)

3

The +3 outside shifts the graph up by 3 (baseline output raised by 3)

⚠️ Common Mistake

Inside changes are opposite to what you think! f(x − 3) shifts RIGHT (not left), f(x + 3) shifts LEFT (not right). It's counterintuitive — memorize it!

💡 How to Describe Any Transformation

Given g(x) = −3f(x) + 2:
→ Reflection over x-axis (negative sign)
→ Vertical stretch by factor of 3
→ Translation up 2 units

Slope-Intercept Form & Linear Modeling

▾
🦁 Coach Ibrahim Says

Slope = rate of change. Szoboszlai covers ~12km a match — that distance per minute is a slope. Trent Alexander-Arnold creates ~3 chances per game — rate per game is a slope. The y-intercept is your starting point before anything happens: Mo Salah has a reputation already at 10 before kickoff, that's your b. And Brahim Díaz? His dribble success rate in Morocco's system starts high from game one — high b-value, steep positive slope. That's a great linear model.

y = mx + b   |   m = slope, b = y-intercept
Slope (m)

Rise over run.
Rate of change.
How steep the line is.

Y-intercept (b)

Where the line crosses the y-axis.
The starting value (x = 0).

📖 Book Example — Reading Slope from Equation

y = −3x − 8

1

Already in y = mx + b form

2

Slope m = −3 (line goes down 3 for every 1 right)

3

Y-intercept b = −8 (line crosses y-axis at −8)

📌 Finding Intercepts from Standard Form (ax + by = c)

x-intercept: Set y = 0, solve for x
y-intercept: Set x = 0, solve for y

📖 Book Example — 2x + 3y = 12
1

x-intercept: set y = 0 → 2x = 12 → x = 6 → point (6, 0)

2

y-intercept: set x = 0 → 3y = 12 → y = 4 → point (0, 4)

3

Plot both points, draw the line ✅

🇲🇦 Morocco — Real-World Model

Morocco's stadium ticket revenue at the Africa Cup: C(x) = 49.5x + 19.5
x = group ticket packages sold, C = total revenue in hundreds of dollars

1

C(4) = 49.5(4) + 19.5 = 198 + 19.5 = $217.50

2

For $465: 465 = 49.5x + 19.5 → 445.5 = 49.5x → x = 9 tickets

💡 Line of Fit vs Line of Best Fit

Line of fit: A line you draw that seems to fit scattered data points (your judgment call).
Line of best fit: Calculated mathematically to minimize total error — the most accurate fit.
Correlation coefficient (r): From −1 to 1. Closer to ±1 = stronger relationship. 0 = no relationship.

Systems of Equations

▾
🦁 Coach Ibrahim Says

A system of equations is like Hakimi and Ziyech making overlapping runs on the right side — the solution is where they meet on the pitch. Sometimes those runs never cross (no solution — parallel lines). Sometimes they're on exactly the same path (infinite solutions). Usually they meet at one exact point, and that's your answer.

Think of Payton Pritchard cutting off a screen to meet a Jaylen Brown drive — if their paths line up perfectly, that's exactly one solution. 🍀 Or picture Ben Seghir making a run while Louza plays a through ball — the exact spot where they connect is the (x, y) solution.

📌 Three Methods
MethodBest WhenHow
GraphingVisual check, estimateGraph both lines, find intersection
SubstitutionOne equation solved for a variablePlug one equation into the other
EliminationCoefficients can cancelAdd/subtract equations to eliminate one variable
📖 Book — Substitution Method

Solve: y = 1 − x and −2x + y = 4

1

First equation is solved for y, so substitute: −2x + (1 − x) = 4

2

−2x + 1 − x = 4 → −3x = 3 → x = −1

3

Plug back: y = 1 − (−1) = 2

4

Solution: (−1, 2) ✅

📖 Book — Elimination Method

Solve: −2x + 2y = −2 and 2x + y = 5

1

Add the equations together: (−2x + 2x) + (2y + y) = (−2 + 5)

2

3y = 3 → y = 1

3

Sub y = 1 into 2x + y = 5: 2x + 1 = 5 → x = 2

4

Solution: (2, 1) ✅

📖 Trickier Elimination — x − 4y = −3 and 4x + y = 5
1

Multiply row 2 by 4: 16x + 4y = 20

2

Add to row 1: (x + 16x) + (−4y + 4y) = (−3 + 20) → 17x = 17 → x = 1

3

Sub x = 1: 4(1) + y = 5 → y = 1

4

Solution: (1, 1) ✅

⚠️ Special Cases — Know These!

No Solution: When you eliminate both variables and get something FALSE like 0 = −20. Lines are parallel (never meet).
Example: 2x + 3y = 5 and 4x + 6y = −10 → multiply row 1 by 2 → 4x + 6y = 10 ≠ −10. No solution ❌

Infinitely Many Solutions: You get something TRUE like 0 = 0. Lines are identical (perfectly overlapping).

💡 Always Check Your Answer

Plug your (x, y) back into BOTH original equations. If both work — you're right. One wrong — recheck your algebra.
